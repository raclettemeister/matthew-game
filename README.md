# Mathieu Range Ta Chambre

Jeu pixel-art : range ta chambre avant que Maman n’arrive.

## Version

**v1.0**

## Lancer le jeu

Ouvre `mathieu-range-ta-chambre.html` dans un navigateur, ou utilise un serveur local (ex. `npx serve .`).

Pour tester uniquement le mode Fortnite en standalone : ouvre `minigame.html`.
